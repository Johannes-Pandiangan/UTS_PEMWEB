Mengenal HTML, CSS, Javascript dan Tailwind

HTML

HTML (Hypertext Markup Language) adalah kerangka atau struktur dasar dari sebuah halaman web. HTML mendefinisikan semua elemen konten seperti judul, paragraf, gambar, dan tautan.

CSS

CSS (Cascading Style Sheets) digunakan untuk menghias atau mendesain halaman web. CSS mengatur tampilan visual, mulai dari warna, font, mempeindah tampilan, hingga tata letak yang responsif.

JavaScript

JavaScript adalah otak di balik interaktivitas web. JavaScript memungkinkan halaman untuk merespons aksi pengguna, seperti validasi form, animasi, konten dinamis, dan interaksi lainnya.

Tailwind CSS

Sebuah framework CSS yang revolusioner. Mempercepat proses desain dengan menyediakan kelas utilitas langsung di dalam HTML tanpa menulis CSS kustom (penulisan biasanya inline).

Tata Cara pembuatan website sederhana

1. Menentukan Tujuan dan Konsep Website

Langkah pertama adalah menentukan untuk apa website akan dibuat.
Tujuan ini akan membantu kamu menentukan isi, tampilan, dan fitur yang akan digunakan.

2. Menyiapkan Struktur Folder dan File

Sebelum menulis kode, buat folder khusus untuk proyek websitemu.
Struktur ini membantu agar file mudah diatur dan tidak bercampur.

3. Membuat Kerangka Halaman (HTML)

Gunakan HTML (Hypertext Markup Language) untuk membuat struktur halaman web.
HTML berfungsi sebagai kerangka atau struktur utama dari sebuah website.

4. Mendesain Tampilan dengan CSS

Gunakan CSS (Cascading Style Sheets) untuk mempercantik tampilan halaman web.
CSS berfungsi untuk mengatur warna, tata letak, font, dan gaya visual lainnya.

5. Menambahkan Interaktivitas dengan JavaScript

Gunakan JavaScript jika ingin menambahkan fitur interaktif, seperti tombol yang bisa diklik, animasi, atau validasi form.
JavaScript adalah otak dari website — membuat halaman bisa “hidup” dan merespons pengguna.

6. Menguji Website di Browser

Setelah semua file dibuat, buka file "index.html" di browser (Google Chrome, Firefox, Edge, dll).
Lalu periksa agar semuanya berfungsi dengan baik (tampilan sesuai, dan semua tombol berfungsi dengan baik).

7. Menyimpan dan Mengunggah ke Internet

Jika website sudah siap, kamu bisa mengunggahnya secara online agar bisa diakses orang lain.
Beberapa layanan gratis yang bisa digunakan adalah Vercel, Netlify, ataupun Github Pages.
Cukup buat akun, lalu upload file web-mu, dan kamu akan mendapatkan alamat websitemu.
